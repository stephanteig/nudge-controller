# Firmware

The controller runs [ZMK](https://zmk.dev) — open-source, Bluetooth-first
keyboard firmware — on a **nice!nano v2** (nRF52840). ZMK gives us BLE 5.0,
multi-device profiles, deep-sleep power management, and rotary-encoder support
with no custom C: just devicetree + keymap config.

## What's in `firmware/`

```
firmware/
├── west.yml                      # ZMK manifest
├── build.yaml                    # build matrix (nice_nano_v2 + shield)
├── nudge_controller.keymap       # bindings + the 4 layers
├── nudge_controller.conf         # BLE, deep sleep, encoder, battery
└── boards/shields/nudge_controller/
    ├── nudge_controller.overlay  # matrix, encoder, pin assignments
    ├── nudge_controller.zmk.yml  # shield metadata
    ├── Kconfig.shield
    └── Kconfig.defconfig
```

> handles BLE, multi-profile pairing, and deep sleep natively.
