# Nudge Controller

**An open source DIY Bluetooth editing controller for DaVinci Resolve — a ~$130 alternative to the $475 Blackmagic Speed Editor.**

## What is this?

Nudge Controller is a 21-key wireless keypad with a weighted jog/shuttle encoder,
built from off-the-shelf parts around a [nice!nano v2](https://nicekeyboards.com/docs/nice-nano/)
running [ZMK](https://zmk.dev) firmware. It pairs over Bluetooth 5.0 with up to 5
devices and works in any application — it's just a keyboard. A companion app lets
you design layouts, import your existing Resolve shortcuts, and get AI layout
suggestions.

You can try the whole layout on an iPad **before building anything**, then order
the parts, assemble the board, and flash the firmware.

## Why build one?

| | Nudge Controller | Blackmagic Speed Editor |
|---|---:|---:|
| Cost | **~$130** | **$475** |
| Works in | Any app | DaVinci Resolve only |
| Bluetooth | ✅ BT 5.0 | ✅ |
| Customisable | ✅ Fully | ❌ |
| Open source | ✅ | ❌ |
