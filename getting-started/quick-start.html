Quick Start

Path A — Test the layout on an iPad

Start the server on your Mac

cd prototype
npm install
npm start

Find your Mac’s IP

ipconfig getifaddr en0

Open the controller on your iPad

Go to https://stephanteig.github.io/nudge-controller/controller , enter the IP when prompted, and tap Connect. Give DaVinci Resolve focus and try the keys.

See Prototype → Setup for details.

Path B — Build the hardware

Order the parts

Use the Bill of Materials (~$130 all-in).

Assemble

Follow the PCB and Case guides.

Flash + pair

See Path C.

Path C — Flash the firmware

Build

Push the firmware/ config to a ZMK config repo; GitHub Actions produces a .uf2. See Firmware → Flashing.

Flash

Double-tap reset on the nice!nano and drag the .uf2 onto the NICENANO drive.

Pair

Pair “Nudge Controller” over Bluetooth and start editing.
