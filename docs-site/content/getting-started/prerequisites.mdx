# Prerequisites

What you need depends on which track you're on.

## Try the prototype
- An **iPad** (or any tablet/phone with a modern browser)
- A **Mac** with [Node.js 18+](https://nodejs.org)
- DaVinci Resolve installed
- Both devices on the **same Wi-Fi network**

## Build the hardware
- Soldering iron + solder, flush cutters, tweezers
- A multimeter (continuity + LiPo polarity checks)
- The parts in the [Bill of Materials](/hardware/bom)
- For the PCB/case: [KiCad 7+](https://www.kicad.org/) and access to a PCB fab
  + 3D printer / laser cutter (or an online service)

## Flash the firmware
- A **GitHub account** (the `.uf2` is built in GitHub Actions)
- A USB-C **data** cable for flashing + charging
- No local toolchain required — ZMK builds in the cloud

## Run the configurator app
- [Node.js 18+](https://nodejs.org)
- (Optional) an **Anthropic API key** for AI layout suggestions
